## 安装
### CocoaPods

在Podfile添加下面的代码：

```ruby
pod 'YQExtensions'
```

然后，运行以下命令：

```ruby
$ pod install
```

### Carthage
在Cartfile中添加下面的代码：


```ruby
github "weixinbing/YQExtensions"
```

编译 framework：

```
$ carthage update
```

如果这是你首次在项目中使用 Carthage，你将需要进行一些额外的步骤，它们在 [Carthage](https://github.com/Carthage/Carthage#adding-frameworks-to-an-application) 中有解释。
